# tech-salary-radar

Self-updating dashboard of the global tech job market — what skills companies hire for and how much they pay. Rebuilds itself every morning for $0/month.

Data comes from the public career-board APIs (ATS) of hundreds of tech companies (Greenhouse, Lever, Ashby). Dashboard is in Russian, aimed at CIS developers: *what to learn and how much it pays.*
